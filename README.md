# ServisI/O 🛠️


**Precision Workshop Management Platform**
*Modern, Secure, and Bidirectional Cloud Sync.*

ServisI/O is a premium multi-platform mobile application designed for modern workshops. Built with Flutter and Clean Architecture, it provides a robust ecosystem for managing inventory, transactions, customers, and staff performance with state-of-the-art security.

---

## 🚀 Key Features

- **Workshop Intelligence**: Real-time revenue tracking and performance statistics.
- **Inventory Ledger**: Full inventory life-cycle management with custom SKU, stock history, and low-stock alerts.
- **Adaptive CRM**: Manage customers and vehicle histories with a professional touch.
- **Shield Protection System**: 
    - **Hybrid Security**: Three-zone security policy for data isolation.
    - **Military Grade Encryption**: AES-GCM 256-bit PII encryption.
    - **Biometric Enclave**: Seamless unlock using hardware-backed biometrics.
- **Omni-Sync Technology**: Bidirectional sync between ObjectBox (local) and Firestore (cloud) with intelligent conflict resolution.
- **Reporting & Receipting**: Generate professional WhatsApp receipts and detailed financial reports.

---

## 🛠️ Technology Stack

- **Framework**: Flutter (Dart)
- **Architecture**: Clean Architecture + Riverpod (Functional DI)
- **Local Database**: ObjectBox (High Performance NoSQL)
- **Cloud Infrastructure**: Firebase (Firestore, Auth, Functions, Storage)
- **Security**: AES-GCM Encryption, Biometric Auth, Session Management.
